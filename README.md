# 🔮 Эзотерический Таро-Бот с Платным Доступом

Этот проект представляет собой полностью автоматизированный Telegram-бот, который служит шлюзом доступа (пейволлом) для вашего веб-приложения раскладов Таро.

## 🏗 Архитектура Проекта
Проект разделен на две независимые части:
1. **Frontend (Визуальная часть/Сайт):** Ваши файлы `index.html`, `script.js`, `style.css`. Идеально завернуты в приложение и выложены на бесплатный хостинг GitHub Pages. 
   Текущая рабочая ссылка (клуб): `https://KeltSham.github.io/TarrotBot/`. 
2. **Backend (Бот-Охранник на Python):** Скрипт `bot.py`, который живет на компьютере или сервере и управляет платежами.

## 💰 Бизнес-модель
Бот полностью сам обслуживает клиентов:
1. Новый клиент нажимает `/start` -> Получает 7 дней **бесплатного триала** и кнопку веб-приложения к сайту.
2. По истечении 7 дней доступ к кнопке расклада блокируется.
3. Бот генерирует красивый Telegram-чек на **250 ⭐️ Звезд** (Telegram Stars) за 30 дней.
4. После успешного платежа через Apple/Google Pay, бот мгновенно накидывает в базу еще **+30 дней** доступа, открывает кнопку и благодарит человека.

## 👑 Панель Администратора (Секретная)
Бот знает, что вы - Владелец, благодаря вписанному в код личному ID (`1491094235`). Только у вас работает специальная команда:
* **`/admin`** — Вызывает меню администратора, где можно:
   1. Читать список всех пользователей, когда они зашли и какие у них статусы оплат.
   2. Вручную бесплатно/в подарок выдавать подписки по ID (`например: 1491094235 365` — выдаст доступ на целый год).
* **Уведомления:** Бот автоматически шлет вам в Личные Сообщения push-уведомление каждый раз, когда кто-то новый нажал Start или кто-то купил подписку.

## 💻 Как запускать бота вручную?
Бот в данный момент работает прямо из вашего редактора VS Code или терминала.
Если бот "ляжет" или вы перезагрузите компьютер, нужно просто снова зайти в эту папку `vacant-sunspot` через Терминал и написать команду:

```bash
python3 bot.py
```
*На заметку: Библиотеки мы уже установили (через `pip3 install pyTelegramBotAPI`), поэтому качать их заново не придется.*

## 💾 База Данных (`users.db`)
В папке появился файл **`users.db`**. Там хранятся все даты подписок. Этот файл ни в коем случае нельзя удалять со своего диска — иначе бот забудет всех, кто ему платил!
